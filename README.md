# clear-nota-to-csv

## setup:

```
yarn
```

## cli usage:

```
Usage: -f <path>

Options:
  --help         Show help                                             [boolean]
  --version      Show version number                                   [boolean]
  -v, --verbose  Log all parsing process                               [boolean]
  -f, --file     Path to file                                [string] [required]
```

## example:

```yarn start:cli -f example.pdf -v```


## http server usage:

```yarn start:server```
